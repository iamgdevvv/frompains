# Frompains
Funny quotes for programmers base on their work.

Website build by HTML5, CSS3, Javascript.

Tech stack using Webpack, Tailwind, Formspark and Lazysizes.


## How to install
```bash
npm install
```

## Run Project
```bash
npm start
```

## Build Project
```bash
npm run build
```
